# Create REST API app using Django Rest Framework 
Tutorial by @BugBytes DRF playlist|https://www.youtube.com/playlist?list=PL-2EBeDYMIbSXhV8FMC1hVD32Fi6e4l2u

Step 1: Create new venv and install required packages as below:
(venv) >> pip install django
(venv) >> pip install djangorestframework
(venv) >> pip freeze > requirements.txt

Step 2: new project and new app in that project
(venv) >> python -m django startproject mysite
** new folder "mysite" will be created
(venv) >> cd mysite
(venv) >> python manage.py startapp api
** "api" app folder will be created

Step 3: Add the "api" app to mysite/settings.py in INSTALLED_APPS.
        Also, add new file "serializers.py" in "api"

Step 4: Create class-based models in api/models.py
class User --> blank for now
            |-> w/ functions in_stock()
class Product --> product_name, description, price, stock, image
class Order --> order_id, user, created_at, status {Confirm, pending, cancelled}, products
class OrderItem --> Order, Product, Quantity 
                |-> w/ functions item_subtotal = product.price x Quantity

Step 5: Create the database tables for new model class using django's migration functions
(venv)mysite >> python .\manage.py makemigrations

        creating new folder management/commands and add files init.py and populate_db.py
